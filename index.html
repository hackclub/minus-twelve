what is this?

Minus Twelve is a You Ship, We Ship event about tools.
Use JavaScript or TypeScript to design a useful piece of software, and we'll send you a shiny new microcontroller board to help you design some useful hardware!

what's a tool?

A tool is something that helps you and other hackers solve a problem. It should be easy for people to use, and it should be usable by lots of different people.

For this YSWS, you should aim to spend 12 hours or less designing your tool. You can spend more time on it if you need to, but the focus is on smaller projects — it doesn't have to be perfect!

Consider the following questions:

  1. What is something I've had to write code for over and over again that can be abstracted to make it easier to do?
  2. Is there something I've had to do in the past that I can improve by solving a certain problem?

If you have an answer for either of these, there's probably a good tool in there somewhere!

Still stuck? Here are a few examples of great tools:

  • tentamen, which helps you test JavaScript code (by @lux)
  • pretty-ms, which helps you make time values look nice (by @sindresorhus)
  • progress, which helps you draw text-based progress bars (by @visionmedia)

what do i need to do?

  1. Create a tool.
    This must be written in JavaScript or TypeScript, but can do anything you want. Be creative!
    It also needs to include a well-written description of how you're supposed to use it. Look at the examples above for ideas!
  2. Make it public.
    Your tool must live in a public GitHub repository so that other hackers can find it and look at your code.
    It also needs to be published to a JavaScript package manager, such as npm.
  3. Tell us about it.
    Join #minus-twelve on the Hack Club Slack and tell us about your tool!
    If you want to hack on your tool with others, we will also be running two huddles in this channel every two days — one at 12PM EST for US/EU and one at 10PM EST for APAC!
  4. Get a microcontroller!
    If you finish creating your tool and submit it by January 27, 2024 at 11:59PM EST, we'll send you a microcontroller to help kickstart your next project!
    We might even send some extra accessories, too. Stay tuned...
