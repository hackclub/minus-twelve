what is this?

Minus Twelve is a You Ship, We Ship event about tools.
Use JavaScript or TypeScript to design a useful piece of software, and we'll send you a shiny new microcontroller board to help you design some useful hardware!

what's a tool?

A tool is something that helps you and other hackers solve a problem. It should be easy for people to use, and it should be easy to apply to some other type of project.

Consider the following questions:

  1. What is something I've had to write code for over and over again that can be abstracted to make it easier to do?
  2. Is there another project I've worked on in the past that I can improve by solving a certain problem?

If you have an answer for either of these, there's probably a good tool in there somewhere!

Still stuck? Here are a few examples of great tools:

  • tentamen, which helps you test JavaScript code (by @lux)
  • pretty-ms, which helps you make time values look nice (by @sindresorhus)
  • progress, which helps you draw text-based progress bars (by @visionmedia)

what do i need to do?

  1. Create a tool.
    This needs to be written in JavaScript or TypeScript, but can do anything you want. Be creative!
  2. Host it on GitHub.
    Your tool should live in a public GitHub repository so that other hackers can find it and look at your code.
    Additionally, the repository should include a well-written description of how your tool is meant to be used.
  3. Tell us about it.
    Join #minus-twelve on the Hack Club Slack and tell us about your tool!
    We will also be running regular huddles in this channel if you want to hack on your tool with others!
  4. Get a microcontroller!
    If you finish creating your tool by the end date (to be announced), we'll send you a microcontroller to help kickstart your next project!
    We might even send some extra accessories, too. Stay tuned...
