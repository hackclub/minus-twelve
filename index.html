what is this?

Minus Twelve is a You Ship, We Ship event about tools.
Design a useful piece of software, and we'll send you a shiny new microcontroller board to help you design some useful hardware!

what's a tool?

A tool is something that helps you and other hackers solve a problem. A good tool is small, easy to use, and can be used by lots of different people.

Some examples made by @lux, the author of this event:

  • tentamen, which helps you test JavaScript code
  • yeow, which helps you parse command-line arguments
  • linn, which helps you standardize CSS rules

what do i need to do?

  1. Create a tool.
    This can do anything you want and be written in any programming language you want. Be creative!
    Stuck? Try to think of something that could work together with something else you've made, or look at the above examples for inspiration!
  2. Host it on GitHub.
    Your tool should live in a public GitHub repository so that other hackers can find it and look at your code.
    Writing a good description in your README file of how to use your tool is also highly encouraged!
  3. Tell us about it.
    Join #minus-twelve on the Hack Club Slack and tell us about your tool!
    We will also be running regular huddles in this channel if you want to hack on your tool with others!
  4. Get a microcontroller!
    If you finish creating your tool by the end date (to be announced), we'll send you a microcontroller to help kickstart your next project!
    We might even send some extra accessories, too. Stay tuned...
